Running by community, for community

Picusly is the first decentralized, blockchain-integrated and community-driven social media network where users pay to use the infrastructure, without a middleman or centralized organization. All infrastructure provides by community members who are rewarded for running servers. We solve cost problem of decentralized applications with this model.

Unique monetization model for node owners

To connect to a server, users pay the PCST token fee set by the person running the server and access the entire network by connecting to the server.

No middleman

Users pay the fee to use the server directly to the wallet determined by the server owner. The server checks the payments on the blockchain behalf of node owner and automatically accepts the users to network.

Democratic Pricing

Node owners set all prices. There is no upper or lower price limit, but the total amount of PCST tokens the node owner will receive from their users per month cannot exceed the number of PCST tokens they have staked. Pricing is calculated using the “price based on the staking” method.
